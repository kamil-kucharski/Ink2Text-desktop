# Sample Notes / Przykładowe notatki

## English

This folder contains sample handwritten note images that can be used to test Ink2Text after running the application from source or from the built `.exe` file.

Import the images named `note1.png`, `note2.png`, `note3.png`, `note4.png`, and `note5.png` into the app, then use the AI note creation feature to check how the application converts handwritten notes into editable text.

These files are included only for testing and demonstration purposes.

## Polski

Ten folder zawiera przykładowe zdjęcia odręcznych notatek, których można użyć do testowania Ink2Text po uruchomieniu aplikacji z kodu albo z gotowego pliku `.exe`.

Zaimportuj do aplikacji zdjęcia `note1.png`, `note2.png`, `note3.png`, `note4.png` i `note5.png`, a następnie użyj funkcji tworzenia notatki przez AI, aby sprawdzić, jak aplikacja zamienia odręczne notatki na edytowalny tekst.

Pliki służą wyłącznie do testów i demonstracji działania aplikacji.
